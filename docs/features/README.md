# Feature Index

Use this directory for feature-level documentation once a feature grows beyond a single controller/DAO/JSP.

## Current Feature Areas

- Authentication and role selection
- Category management
- Product management
- Admin dashboard
- Product JSON API

## Future Feature Areas

- POS checkout
- Inventory and warehouse management
- Orders and invoices
- Customers
- Suppliers
- Payments
- Finance transactions
- Reports

Each feature document should link its controllers, DAOs, models, JSPs, API routes, database tables, protected-module impact, and planning files.
